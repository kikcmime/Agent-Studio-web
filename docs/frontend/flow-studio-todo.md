# Flow Studio TODO

## 目标

把当前 `Flow Studio` 从静态展示页改造成真正可编排的多 Agent 工作流编辑器。

核心方向：

- 前端使用 React Flow
- 支持节点拖拽、连线、选中、删除
- 支持右侧节点配置
- 支持 Agent 节点绑定已有 Agent
- 后续对接后端 Agno 执行

## 当前问题

- 现在的中间画布还是静态占位，不可编排
- 节点不能真实拖动
- 连线不能编辑
- 节点配置还是假数据
- 没有真正的 Flow 保存结构

   - End
4. 左侧节点库支持添加节点
5. 支持节点选中
